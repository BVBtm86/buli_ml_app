# Machine Learning Application

Machine Learning represents the ability of a computer to learn from studying data and statistics. It analyses data and learns to predict the outcome.

App Features

* Unsupervised Learning 
  - PCA Analysis
  - Hierarchical Clustering
  - K-Means Clustering
* Supervised Learning 
    1. Classification
       - Logistic Regression
       - SVM
       - Naive Bayes
       - Decision Tree
       - Random Forest
       - XgBoost
    2. Linear Regression
       - Logistic Regression
       - SVM
       - Naive Bayes
       - Decision Tree
       - Random Forest
       - XgBoost

Data: is based on the Bundesliga 2017-2022 Seasons Team Statistics
